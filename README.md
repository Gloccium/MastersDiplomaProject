# Causal Inference MVP: SDID vs BSTS
## Репозиторий к магистерской диссертации на тему:
**«Оценка причинного эффекта продуктовых изменений в условиях невозможности проведения A/B-тестов: сравнение Synthetic Difference-in-Differences и Bayesian Structural Time Series».**

Данный проект представляет собой программный комплекс для оценки причинно-следственных связей (Causal Inference) на панельных данных и временных рядах. Включает в себя конвейер генерации синтетических данных (DGP), кастомную реализацию алгоритма SDID, интеграцию байесовских моделей BSTS и интерактивный дашборд на базе Streamlit.
## Структура проекта
Проект построен по модульной архитектуре:

* src/ — Вычислительное ядро (Core):
  * dgp.py — Генератор синтетических данных. Позволяет имитировать метрики e-commerce: задавать авторегрессионный шум, расходящиеся тренды и сложную гетерогенную сезонность.
  * models.py — Классы-обертки алгоритмов:
    * BaselineDiD — Классический OLS Difference-in-Differences.
    * CausalImpactBSTS — Байесовские структурные временные ряды (адаптировано для работы с новыми версиями pandas).
    * SyntheticDiD — Практическая реализация алгоритма SDID с использованием L2-регуляризованной квадратичной оптимизации (SLSQP).
* app/ — Интерактивное веб-приложение (MVP):
  * main.py — Точка входа Streamlit-приложения (пользовательский интерфейс).
  * olist_utils.py — Инженерный конвейер (ETL) для обработки сырых реляционных данных бразильского маркетплейса Olist (очистка, джоины, формирование сбалансированной панели).
* notebooks/ — Исследовательская среда:
  * experiments.ipynb — Jupyter-ноутбук с симуляциями Монте-Карло для статистического сравнения моделей (расчет Bias, RMSE, Coverage Rate, Inference Time).
* data/ — Директория для хранения сырых данных

## Инструкция по запуску

Убедитесь, что у вас установлен Python 3.9 или выше. Рекомендуется использовать виртуальное окружение:

**Клонирование репозитория**
* git clone https://github.com/Gloccium/MastersDiplomaProject.git
* cd MastersDiplomaProject

**Создание и активация виртуального окружения (Windows)**
* python -m venv .venv 
* .venv\Scripts\activate

**Для macOS/Linux:**
* python3 -m venv .venv
* source .venv/bin/activate

## Установка зависимостей
pip install -r requirements.txt

**Примечание**: 

Алгоритм BSTS (библиотека pycausalimpact и PyTensor под капотом) может требовать наличия C++ компилятора (например, MSYS2/MinGW на Windows) для максимальной скорости работы.

## Воспроизведение гипотез H1, H2, H3

Чтобы запустить симуляции Монте-Карло, подтверждающие гипотезы диссертации (H1, H2, H3):

1. **Запустите Jupyter Notebook:** jupyter notebook
2. Откройте файл notebooks/experiments.ipynb
3. Запустите все ячейки (Run All). Скрипт проведет 50 итераций генерации данных и выведет в консоль итоговые усредненные метрики смещения (Bias), покрытия (Coverage) и скорости (Inference Time).

## Запуск аналитического веб-приложения (Streamlit MVP)

Для запуска интерактивного веб-сервиса выполните команду в корневой папке проекта:

streamlit run app/main.py

После этого в вашем браузере автоматически откроется страница http://localhost:8501.

В приложении вы можете:

* Сгенерировать синтетические данные с различными паттернами (тренды, сезонность).
* Выполнить ETL-пайплайн для данных Olist, выбрав интересующие вас даты и штаты.
* Рассчитать причинный эффект выбранной моделью (DiD, SDID, BSTS) и изучить интерпретируемые графики контрфактических сценариев.
